Why Sentinel Route Outperforms Existing Models

  • Only system combining NLP keyword extraction + temporal scoring + haversine routing in a single pipeline
  • Adjusts safety scores dynamically based on time of day — no existing free routing tool does this
  • 15-point deduction model is fully transparent and explainable — unlike black-box ML models
  • Does not require expensive labeled training data — uses unsupervised keyword matching
  • Supports dual-route comparison (fastest vs safest) — helping users make informed decisions
  • Specifically designed for vulnerable demographics (solo travelers, women, elderly)
  • Open dataset driven — works offline without external API dependencies
